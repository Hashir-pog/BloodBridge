# BloodBridge - Blood Donation Management System

BloodBridge is a blood donation management platform that connects donors, recipients, blood banks, and administrators.

## Features
- User authentication
- Donor and recipient management
- Blood request handling
- Donation management
- Blood bank management
- Notifications and feedback

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQL/database concepts

## How to Run
```bash
cd backend
npm install
npm start
```

Open the frontend/index.html file in a browser.
